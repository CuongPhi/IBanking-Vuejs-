## Quick start

## :cloud: Build Setup

### install dependencies
`npm install`
### serve with hot reload at localhost:8080
`npm run dev`
### build for production with minification
`npm run build`

- Clone the repo: `git clone https://github.com/CuongPhi/IBanking-frontend.git`.
